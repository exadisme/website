> latest blog post

Jan 30th, 2022

I spent the day learning about some new ways to style my webpage using CSS and that is with the help of the flex attribute! As of time of writing, approximately 1pm EST, I only have a layout that plays nice with a higher desktop resolution. It took me a while to properly grasp how to use flex but I think it's a nice way to arrange a decent mix of various columns/width boxes. I wanted to make sure the blog post section, at least on a larger screen, has enough height that one could read an entire post, or at least most of the post, without having to do much scrolling.

I also learned about importing font-awesome icons which is insanely handy. When I started, I had the mindset that I would be designing everything from scratch but it seems, with the help of various tools and frameworks, I really only have to adapt existing styles/designs to suit my needs.

CSS is extremely handy for styling pages without having to alter the underlying html however, as there are so many different devices and layouts, you're essentially creating multiple designs to target specific device sets. This means designing a webpage essentially requires that you design the same page multiple times. I'll have to get used to creating designs before hand because up until now I've just been making it up as I go. I can definitely see how having design software with preset heights and widths for specific devices can come in handy.
